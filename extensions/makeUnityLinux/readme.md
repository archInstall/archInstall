### makeUnityLinux

Installiert ein Arch Linux mit lightdm als Display-Manager und Unity als
Window-Manager/Arbeitsumgebung. Es werde auch andere Ubuntu-Typische Pakete,
wie Libre-Office oder Thunderbird installiert.
